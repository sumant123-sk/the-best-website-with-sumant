1. HTML

HTML (HyperText Markup Language) is used to structure web content. Think of it as the skeleton of a website.

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

2. CSS

CSS (Cascading Style Sheets) is used for styling HTML elements. It's like the skin, clothes, and makeup for the skeleton, making it look beautiful.

body {
  background-color: #121212;
  color: white;
  font-family: Arial, sans-serif;
}
p {
  color: yellow;
  font-size: 18px;
}

3. JavaScript

JavaScript is used to make web pages interactive. It's the brain that adds functionality, like animations, buttons, and user interaction.

<script>
  document.addEventListener("DOMContentLoaded", function() {
    console.log("The page is fully loaded!");
    // You can add more interactive logic here.
  });
</script>
